# configuration

## config.ini
slack 설정. slack api token, channel id
정원사들 설정. 시작일, 진행일
db설정. database, host, port

를 설정합니다.

예제.
```
; config.ini
[DEFAULT]
SLACK_API_TOKEN = xoxb-...
CHANNEL_ID = C01...

START_DATE = 2021-01-18
GARDENING_DAYS = 100

[MONGO]
DATABASE = garden6
HOST = localhost
PORT = 27017
```

## users.yaml

```
github organization:
  slack: slack name
```
형태로 입력합니다.

slack name은 https://api.slack.com/methods/users.list/test 에서 확인할 수 있습니다.
